DETOUR - AN AUTOMATED ANALYSIS ON SIMULATION OF TRAFFIC SITUATION THROUGH VIDEO PROCESSING WIT SHORTEST ALTERNATIVE PATH

Jeffrey D. Canoy, Loren E. Narciso,
Randy S. Gamboa


ABSTRACT

  The researchers recognized the growing traffic congestion and the problem it caused the people. This led the researchers to conduct this study. The study simulated a traffic situation using the videos downloaded from the Internet and from the Public Security and Safety Command Center of Davao. The study analyzed the traffic videos using image processing and classified the video’s traffic condition as light, moderate and heavy by obtaining its average speed and verge density. The algorithms that were used in processing the video were Gaussian Mixture Model (GMM) for background modeling, Blob Analysis for blob and blob features extraction and Kalman filter for object tracking. The researchers used geographic information system for the system’s mapping features and Dijkstra’s Algorithm in finding the alternative path. The system was developed using MATLAB. The accuracy of the study was 85.38%.
