ENHANCEMENT OF THE DAVAO COLLEGES AND UNIVERSITIES NETWORK

Cindy Canang-Moldes,
Maureen M. Villamor


ABSTRACT

  This study was intended for the enhancement of the Davao Colleges and Universities Network (SACUN) Library Web Portal. Part of the objectives of this study were to: design and develop Web Portal; be able to post news, updates and other relevant library-related information; ass an all member schools websites with RSS feeds; and add a forum. This study used the Web Portal Development Strategy that included portal initiation, portal analysis and design, and testing and development.

  The design of a library Web Portal using WordPress as a publishing platform proved to be a suitable choice because its user-friendly features. The portal is found to be a very good information tool that is well suited in this fast paced digital environment.

   The DACUN Library Web Portal is a good venue for DACUN member institutions not only in posting and providing relevant news and updates but also an avenue for sharing of innovative ideas and promoting of activities. Furthermore, the promotion and marketing of DACUN activities and events is made possible through an event organizer feature with report generation. Updates to different DACUN member schools an be continually disseminated via RSS feeds, Forums and discussions enable also participation, camaraderie and interaction among DACUN members.
